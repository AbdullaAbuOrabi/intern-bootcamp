# FAQ

## What is this project?
This project is an internship bootcamp project for learning data, analytics, machine learning, and GenAI concepts.

## What data is used?
The project uses synthetic e-commerce data including customers, products, orders, order items, and transactions.

## Is the data real?
No. The data is synthetic and created for learning purposes.

## What is the purpose of the data?
The data will be used for SQL practice, analytics, machine learning, and RAG experiments.
