# Prompt Templates in LangChain

Prompt templates are reusable prompt structures that help developers communicate with large language models in a clear and organized way.

Instead of writing the same prompt again and again, developers can create a template with variables.

For example, a template can include variables such as user question, context, document text, or product description.

LangChain makes this easier by providing PromptTemplate objects.

Prompt templates are useful because they make LLM applications more consistent, reusable, and easier to automate.
