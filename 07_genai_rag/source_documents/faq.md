# Project FAQ

## What is RAG?

RAG stands for Retrieval-Augmented Generation. It allows an AI system to answer questions using external documents.

## Why do we use chunking?

Chunking splits large documents into smaller parts so the system can retrieve only the most relevant information.

## What is a vector store?

A vector store saves document chunks as embeddings so similar information can be searched quickly.

## What is semantic search?

Semantic search finds text based on meaning, not only exact keywords.
